Projeto port.h

O projeto porth.h é uma biblioteca open source desenvolvida em C++ para permitir a compilação de códigos em uma linguagem semelhante ao Portugol, porém com uma extensão ao paradigma orientado a objetos, ela foi criada com o objetivo de facilitar o ensino de programação.

FMDS Digital

Aplicativo web pioneiro no gerenciamnto remoto do chão de fábrica.

O FMDS digital foi desenvolvido e implantado na unidade Pirapora da VLI logística para realizar o gerenciamnto do chão de fábrica de forma remota em virtude da pandemia do COVID-19.

Spring Blockchain

Blockchain desenvolvida utilizando o framework Spring Boot e Javascript.

O projeto Spring Blockchain consiste em uma implementação de Blockchain desenvolvida utilizando a linguagem Java por meio do framework Spring Boot. O grande diferencial desse projeto é a possibilidade de utilização do próprio poder de processamento dos clientes (smartphones,tablets, pc's) para a mineração dos blocos.

Linguagem SOL

Uma linguagem de programação para brasileiros desenvolvida por brasileiros!

A linguagem SOL é uma linguagem de programação baseada em portugol e em C++ desenvolvida por Sebastião Fortes Avelar de Almeida com o objetivo de permitir que pessoas de fora do mundo da programação possam criar pequenos programas por meio de um código simples e de fácil compreensão, essa linguagem também tem o objetivo de facilitar o ensino de programação para iniciantes.

Narrador da Web

Desenvolvido para pessoas com baixa visão, a extensão realiza a leitura em voz do conteúdo dos sites

Extensão desenvolvida para facilitar a navegação na web por parte de pessoas com baixa visão. O Narrador da Web realiza a leitura em voz do conteúdo de sites da web, basta o usuário posicionar o cursor do mouse sobre textos e links que o conteúdo será magicamente narrado.
